The DATA extraction/processing aspect of the Zeitgeist project.
Built in python, it'll supply the front-end with relevant data.


# Libraries Installed:
## core webscraping
- "pip install requests"
- "pip install feedparser"
- "pip install playwright" 
- "pip install pytest-playwright"   // end-to-end testing plugin
- "playwright install"              // to finalize installation (installs browser binaries)
- "pip install fastapi uvicorn"     // http server API-establishing libraries  


## core NLP and ML libraries
- "pip install -U spacy"                    // Natural language processing library/api 
- "python -m spacy download en_core_web_sm" // English tokenizer, tagger, parser and NER
- "pip install scikit-learn"                // Simple and efficient tools for predictive data analysis


- “pip install anthropic” for LLM API calls for abstract computations


## Utilities 
- "pip install pandas"  // fundamental high-level building block for doing practical, real world data analysis 
- "pip install numpy"   // scientific computing with Python
